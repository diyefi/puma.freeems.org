Descripción detallada

Routines for controlling a hd44780-compatible lcd display.

This file holds the required functions to initialize, read and write an lcd. It must be connected using a 8 bit wide data bus, and connecting all the control lines, including R/W. Every high level function will return once the lcd has completed the task.
