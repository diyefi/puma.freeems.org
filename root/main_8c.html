Descripción detallada

Aquí está el centro del sistema.

Este archivo contiene las funciones de mas alto nivel -control de temperatura y el lazo principal-

El sistema, a grandes razgos, realiza lo siguiente:

Con el timer 2 vigila si el boton PROG fue pulsado durante un seg, en caso afirmativo habilita el modo programacion (en main() ), y muestrea el estado de los botones, manejando los PaP.

Además, muestrea continuamente la temperatura, tension y corriente en el ISR del SPI,el dato es depositado en ad_temp_atomic, ad_volt_atomic y ad_amp_atomic.

main() calcula y muestra tensiones y corrientes.

EL modulo 1 del PCA genera el pwm en forma continua para controlar la velocidad del motor.

void main ( void   ) 

Inicializa el sistema y ejecuta el lazo principal.

La inicialización consiste en configurar los módulos integrados en el microcontrolador, entre ellos timers, pwm, y comunicaciones. De esta forma, se disparan todos los mecanismos para que el sistema siga funcionando a partir de las interrupciones generadas por el entorno. Una vez inicializado, se escribe en el display y empieza el lazo principal, que se limita a controlar la temperatura y escribir la tensión y corriente en el display.

void temp_ctrl ( float  temperatura  ) 

Toma medidas contra excesos de temperatura e invoca al pid.

Si la temperatura actual supera el máximo establecido -extraído de la eeprom-, se encienden las alarmas, y se apaga la fuente. Con una simple histéresis, la fuente se enciende cuando se han alcanzado valores seguros de temperatura (90% del máximo).

Una vez chequeada la temperatura, se procede al cálculo de la velocidad del cooler con el lazo pid

Parámetros:
temperatura Temperatura actual
